int isspace ( int  c)

Vérifie si l'on a un caractère blanc, d'espacement. Il s'agit de : espace, saut de page (form-feed, '\f'), saut de ligne (newline, '\n'), retour chariot (carriage return, '\r'), tabulation horizontale ('\t'), et tabulation verticale ('\v').

Paramètres
cle caractère à vérifier.
Renvoie
vrai si le caractère est d'espacement.
